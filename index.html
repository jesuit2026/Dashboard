Datenquelle

Normaler OneDrive-Freigabe-Link zur mobile_screen.json (der 1drv.ms/…-Link reicht — die App macht daraus automatisch einen Direkt-Link). Die Auswahl kommt vom PC, die Kurse holt die App selbst live. Die Watchlist (mobile_watchlist.json) wird automatisch von derselben Quelle mitgeladen — kein zweiter Link nötig.

Aktualisierung

Kurse werden alle 60 Sekunden aktualisiert, solange die App offen ist. Ziehe die Liste nach unten zum manuellen Neuladen.

Alarme & Watchlist bearbeiten

Um Alarme/Ticker direkt am Handy anlegen zu können, brauchst du ein GitHub-Token, das nur für dieses eine Repo Schreibrechte hat:

  1. github.com/settings/personal-access-tokens/new öffnen
  2. Repository access → Only select repositories → Dashboard
  3. Permissions → Contents → Read and write
  4. Token erzeugen und hier einfügen
